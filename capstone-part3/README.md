This project is a Weather Application built using React, TypeScript, and Tailwind CSS.
The goal of the application is to display real-time weather information in a simple, human-friendly way, helping users quickly understand how the weather may affect their day. Unlike traditional weather apps that only present raw data, this project focuses on clarity, readability, and user experience. 

Project Objectives:
-Learn the fundamentals of React with TypeScript
-Practice building and structuring reusable components
-Work with external APIs using fetch
-Handle loading, error, and empty states
-Style a React application using Tailwind CSS
-Build a clean, responsive, and user-friendly interface
